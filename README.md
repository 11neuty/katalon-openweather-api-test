Overview
This project contains API automation testing using Katalon Studio for OpenWeather API.
The test covers weather forecast and air pollution endpoints with validation on response body, JSON schema, and key attributes.

Scope

* 🌤 Get 5-day weather forecast (Jakarta Selatan coordinates)
* 🌫 Get current air pollution

Validation Coverage

* ✅ Response status code
* ✅ Response body structure
* ✅ JSON schema validation
* ✅ Important attributes validation

  * AQI range (1–5)
  * Humidity range (0–100)
  * Pollution components (CO, PM2.5, PM10)

Project Structure

* 📁 Object Repository
  Contains API request definitions

* 🧪 Test Cases
  Weather

  * TC_W_001_Get5DayForecast
  * TC_W_002_ValidateStructure
  * TC_W_003_DataConsistency

  Air Pollution

  * TC_A_001_GetAirPollution
  * TC_A_002_ValidateAQI
  * TC_A_003_ValidateComponents

* 📦 Test Suites

  * TS_OpenWeather

* 📄 Include/schema
  JSON schema files for validation

  * forecast_schema.json
  * air_schema.json

* 📊 Reports
  Contains execution results generated by Katalon

* 📄 report-generator
  Python script to generate custom PDF report from CSV result

Prerequisites

* ⚙️ Katalon Studio
* 🔑 OpenWeather API Key

Setup

1. Open the project in Katalon Studio
2. Go to Profiles
3. Set the following variables:

   * apiKey
   * lat
   * lon

Example for Jakarta Selatan : 
lat = -6.2615
lon = 106.8106

How to Run

1. Open Test Suites
2. Select TS_OpenWeather
3. Click Run

Report

Default Report
Generated automatically after execution:
Reports/TS_OpenWeather/<timestamp>/

Contains:

* 📄 HTML report
* 📄 CSV report

Rate Limit Handling
⚠️ The test minimizes API calls and avoids excessive execution to stay within OpenWeather free tier limits.
